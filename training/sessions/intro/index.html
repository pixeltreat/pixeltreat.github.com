02 Define (UXD)

Information Architecture

  • Inventory All Existing Content
  • Execute A Content Audit
  • Design New Site Architecture
  • Test with users using card sorting / interactive prototypes
  • Iterate based on the user feedback

User stories

  • 'who', 'what' and 'why' of user and business requirements in an easy-to-read narrative
  • As a (type of user)
  • I want (some goal)
  • so that (some reason)

Page template design
(Wireframing)

  • What content should be available on each page
  • Design content structure, layout & interaction patterns
  • Share prototypes with users and ask for feedback
  • Share and explain designs often with the entire team, so they understand every layout/interaction pattern, which helps in finding feasibility issues from development standpoint.

03 Design (UID)

Basic Elements of Visual Design

  • Color palette color theory examines how various choices psychologically impact users.
  • Typography refers to which fonts are chosen, their size, alignment, color, and spacing
  • Shapes every object is composed of shapes
  • Texture depending on how a texture is applied, it may be used strategically to attract or deter attention.

Visual Design Principles

  • Visual design must strike a balance between unity and variety to avoid a dull or overwhelming design.
  • Hierarchy shows the difference in significance between items.
  • Contrast focuses on making items stand out by emphasizing differences in size, color, direction, and other characteristics.
  • Similarity refers to creating continuity throughout a design without direct duplication.
