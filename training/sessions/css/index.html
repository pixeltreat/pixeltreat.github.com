01 Introduction to CSS

What is CSS?

CSS stands for Cascading Style Sheets, it allows web developers to define the look and feel of the Web pages and separate presentation layer from the structure (HTML markup).

The presentation layer defines how the content will appear on screen/print. So CSS keeps content untouched, as content can also be converted to synthetic speech for users who have impaired vision or reading difficulties.

CSS Syntax

A CSS rule has two main parts: a selector, and one or more declarations

CSS Syntax

A CSS declaration always ends with a semicolon, and declaration groups are surrounded by curly brackets.


p {
    color:red;
    text-align:center;
}

Add Styles to HTML?

There are three ways of providing styling information for the Web browsers.

  • Link element Linking external style sheet
  • Style element Embed styles in the head of the document
  • Style attribute Using style attribute adding inline styles

External Styles

An external style sheet is ideal when the styles are applied to many pages. With an external CSS, we can update presentation layer of entire Web site by changing one file.


<head>
   <link rel="stylesheet" type="text/css" href="example.css">
</head>

Internal Styles

Internal styles should be used when a single document has a unique styles.


<head>
    <style type="text/css">
        p { color: red; font-size:120%; }
    </style>
</head>

Inline Styles

An inline style loses many of the advantages of style sheets by mixing content with presentation, which is not recommended so use this method sparingly!


<p style="color: red; font-size:120%;">
    This is a paragraph
</p>

CSS Media Types

  • Media Types allow us to specify how documents will be presented in different media.
  • The document can be displayed differently on the screen, on the paper, with an aural browser, etc.
  • The @media rule allows different style rules for different media in the same style sheet.

CSS Media Types


<head>
    <style>
        @media print {
            p { font-size : 12pt; }
        }

        @media screen {
            p { font-size : 100%; }
        }

        @media tv{
            p { font-size : 140%; }
        }
    </style>
</head>

CSS Media Types

Media Type Description
All Used for all media type devices
Aural Used for speech and sound synthesizers
Braille Used for braille tactile feedback devices
Embossed Used for paged braille printers
Handheld Used for small or handheld devices
Projection Used for projected presentations, like slides
Tty Used for media using a fixed-pitch character grid, like teletypes and terminals

CSS Comments

Comments are used to explain your code, and may help you when you edit the source code at a later date. Comments are ignored by browsers.

Universal selector

This will select all the elements in the document.


*{
    color: red;
    font-size:120%;
}

Type selector

This will select the defined HTML elements in the document.


p{
    color: red;
    font-size:120%;
}

Descendant Selector

This will select the descendant elements


<h1>
  Headline with <em>special</em> text
</h1>

h1 em{
    color: red;
}

ID selector

This will select the element with the ID defined.


<h1 id="heading">
  Simple headline
</h1>

#heading{
    color: red;
}

Class selector

This will select the elements with the class defined.


<h1 class="heading">
  Simple headline
</h1>

.heading{
    color: red;
}

Pseudo-Classes

This will select elements based on some characteristic other than the element name, attributes or content.

Selector Description
a:link, a:visited, a:active, a:hover Selects all unvisited, visited, active and on mouse over links
input:focus Selects the input element which has focus
p:first-letter Selects the first letter of every <p> element
p:first-line Selects the first line of every <p> element
p:first-child Selects every <p> elements that is the first child of its parent

p:after Insert content after every <p> element

Attribute selector

This will select the elements with the attribute defined.

Child selector

This will select the element which is direct child.

Sibling selector

This will select the element which is next sibling.

  <p>sample text</p>

Group selector

This will select all the elements which are comma separated


h1, h2, p {
    color: red;

03 CSS2 Properties

The CSS Box Model

The CSS box model is essentially a box that wraps around HTML elements, and it consists of margins, borders, padding, and the actual content.


div {
    width: 250px;
    padding: 10px;
    border: 5px solid gray;
    margin: 10px;
}

Total element width = width + left padding + right padding + left border + right border + left margin + right margin

Total element height = height + top padding + bottom padding + top border + bottom border + top margin + bottom margin

CSS Colors

CSS colors are defined using a hexadecimal (hex) notation for the combination of Red, Green, and Blue color values (RGB). The lowest value that can be given to one of the light sources is 0 (hex 00). The highest value is 255 (hex FF).

With CSS, a color is most often specified by:

  • HEX value - "#ff0000"
  • RGB value - "rgb(255,0,0)"
  • color name - "red"

CSS Units - Absolute

The absolute units are fixed in relation to each other and anchored to some physical measurement. They are mainly useful when the output environment is known.

Unit Definition
cm centimeters
mm Millimeters
in inches; 1in is equal to 2.54cm
px pixels; 1px is equal to 1/96th of 1in
pt points; 1pt is equal to 1/72nd of 1in

CSS Units - Relative

The relative units specify a length relative to another length. Style sheets that use relative units can more easily scale from one output environment to another.

vmin minimum of the viewport's height and width

CSS Outlines

An outline is a line that is drawn around elements (outside the borders) to make the element "stand out". However, the outline property is different from the border property.

The outline is not a part of an element's dimensions; the element's total width and height is not affected by the width of the outline.


    outline: outline-color outline-style outline-width;

p{
    outline:#00FF00 dotted thick;
}

CSS Margin

The margin clears an area around an element (outside the border). The margin does not have a background color, and is completely transparent. The top, right, bottom, and left margin can be changed independently using separate properties. A shorthand margin property can also be used, to change all margins at once.


    margin: top right bottom left;

p{
    margin-top:100px; margin-bottom:100px; margin-right:50px; margin-left:50px;
    margin: 100px 50px;
}
It is also possible to use negative values, to overlap content.

CSS Padding

The padding clears an area around the content (inside the border) of an element. The padding is affected by the background color of the element. The top, right, bottom, and left padding can be changed independently using separate properties. A shorthand padding property can also be used, to change all paddings at once.


    padding: top right bottom left;

p{
    padding-top:100px; margin-bottom:100px; margin-right:50px; margin-left:50px;
    padding: 100px 50px;
}

CSS Display

The display property defines how a certain HTML element should be displayed.


div{
display: [ block | inline | inline-block | inline-table | list-item | run-in | table | table-caption | table-cell | table-column | table-column-group | table-footer-group | table-header-group | table-row | table-row-group | none | inherit ];
}
The values "inline-table", "table", "table-caption", "table-cell", "table-column", "table-column-group", "table-row", "table-row-group", and "inherit" are not supported in IE7 and earlier. IE8 requires a !DOCTYPE. IE9 supports the values.

inline This is default. The element is displayed as an inline element
inline-block The element is placed as an inline element, but it behaves as a block element
inline-table The element is displayed as an inline table
list-item The element is displayed as a list-item, which means that it has a bullet in front of it
table The element is displayed as a table
table-caption The element is displayed as a table caption

CSS position

The position property specifies the type of positioning method used for an element


p{
    position: [ absolute | fixed | relative | static | inherit ]
}
Value Description
absolute The element is positioned relative to its first positioned (not static) ancestor element
fixed The element is positioned relative to the browser window
relative The element is positioned relative to its normal position
static Elements renders in order, as they appear in the document flow. This is default.
inherit The value of the position property is inherited from the parent element

CSS position

The position property specifies the type of positioning method used for an element

CSS Float

With CSS float, an element can be pushed to the left or right, allowing other elements to wrap around it. Float is very often used for images, but it is also useful when working with layouts. The elements after the floating element will flow around it.


p{
    float: [left | right | none | inherit]
    clear: [left | right | both | none | inherit ]
}

visibility

The visibility property specifies whether or not an element is visible.


p{
    visibility: [ visible | hidden | collapse | inherit ]
}
