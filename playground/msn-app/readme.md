# MSN Application
FED Stack: Angular, SCSS, Gulp and Webpack

## Requirements ##
    Node.js v.4.4.4 and above

## Install ##
    'npm run setup'

## Local Build and Server ##
    'npm run dev'
    
## Production Build ##
    'npm run publish'
    (To Minify JS and CSS, Optimize Images for production)

## Production Build with Server ##
    'npm run serve'
